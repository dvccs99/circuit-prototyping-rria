# Miniprojetos - Prototipagem de Circuitos - CIn/UFPE 2024 - RRIA
### Alunos: Daniel Salvador e Laura Luna

Esse repositório contém os miniprojetos da cadeira de prototipagem de circuitos, ministrada durante a terceira turma residência em robótica e inteligência artificial do centro de informática da UFPE.

## Miniprojeto 1

- O projeto consistia de utilizar um Arduino Uno e componentes diversos para construir um equipamento simulando o alarme de uma casa. A programação deveria ser feita seguindo o padrão de máquina de estados.
- Para rodar este projeto, basta executar o arquivo `mini_projeto1\alarme\alarme.ino`.

## Miniprojeto 2

- Nesse miniprojeto, deveríamos utilizar um ESP32 para refazer o sistema de alarmes, mas dessa vez utilizar também as funcionalidades de comunicação WiFi da placa para gerar avisos para o usuário.
- Para mais detalhes de como rodar, veja o `README.txt` dentro da pasta `mini_projeto2`.


Os projetos foram feitos e testados na Arduino IDE.
