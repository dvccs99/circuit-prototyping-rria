Mini-projeto 2 da cadeira de prototipagem de circuitos da 3º turma da residência em
robótica e IA do CIn. Protótipo de um alarme residencial. O uníco arquivo funcional 
é o alarme_v0.ino que é bastante semelhante ao código do projeto 1, com excecão do motor.

O código do alarme_v2.ino seria o código final, no entanto tivemos muitos problemas para
terminar. Existem escritos nele muitas funcionalidades que demonstram nossa tentativa.

O código teste_server.ino funciona com apenas o ESP32 conectado. Serve para demonstrar o 
código do arquivo alarme_server.py (basta rodar este para ligar o servidor). O banco_de_dados.txt
contém apenas um demonstrativo de usuário e senha que seria usado pelo servidor.
